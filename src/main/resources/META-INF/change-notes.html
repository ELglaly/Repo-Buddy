1.0.4 - Marketplace Fix & Compatibility Expansion

  • Fixed plugin description not appearing on JetBrains Marketplace.
  • Expanded IDE compatibility range: supported from IntelliJ IDEA 2023.2 with no upper build limit.

1.0.3 - UI Polish and Marketplace Presentation

  • Refined the Repository Usage table with cleaner badges, improved typography, and a more polished layout.
  • Redesigned the Call Chain Tracer tree and the repository runner popup for a more professional, readable presentation.
  • Updated the Marketplace plugin description and metadata wording for a clearer, stronger overview page.

1.0.2 - Repository Filtering & Endpoint Search

  • Current File filter - new toggle in the Repository Usage tab. Open any repository file (e.g. UserRepository.java) and click Current File to scope the table to that repository's methods instantly.
  • Endpoint search in Call Chain Tracer - live filter field above the endpoint dropdown. Type an HTTP verb (e.g. POST) or a path fragment (e.g. /users) and the dropdown narrows in real time.
  • Full API path display - endpoint paths now include the controller-level @RequestMapping prefix (e.g. /api/v1/users/{id}).
  • Cleaner endpoint combo display - shows only the HTTP verb and path ([GET] /api/users) without the controller class name.

1.0.1 - Stability & Build Improvements

  • Enhanced build configuration and random data generation strategies.
  • Updated settings and plugin files for improved documentation and versioning.

1.0.0 - Initial Release

  • Run gutter actions on Spring Data repository methods.
  • Live execution via embedded Java agent.
  • SQL capture via Hibernate StatementInspector.
  • Smart parameter form with per-type widgets.
  • Random data generation with field-level and generate-all actions.
  • Pageable and enum-aware parameter handling.
  • Call Chain Tracer with summary insights and source navigation.
  • Repository Usage table with filtering and export support.
  • Automatic run-configuration patching on project open.
  • Theme-aware UI for light and dark IntelliJ themes.
